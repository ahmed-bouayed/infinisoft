# Infini-Soft – Site Vitrine

Ce projet est le site vitrine officiel d'**Infini-Soft**, une société spécialisée dans les services informatiques. Le site présente l'ensemble des expertises et solutions proposées par l'entreprise.

## 🚀 Technologies utilisées

- **HTML5** & **CSS3** (Variables CSS, Flexbox, Grid)
- **JavaScript** (Vanilla JS)
- **Google Fonts** (Syne & DM Sans)
- Architecture modulaire basée sur des sections HTML dynamiques.

## 📂 Structure du projet

Le projet est organisé comme suit :

- `index.html` : Point d'entrée principal de l'application.
- `sections/` : Contient les différents modules de pages (Accueil, À propos, Services, Contact, etc.).
- `css/` : Feuilles de style, incluant le support du mode sombre/clair.
- `js/` : Logique JavaScript pour la navigation et l'interactivité.
- `img/` : Ressources graphiques et logos.

## 🛠️ Installation et Utilisation

1. Clonez le dépôt :
   ```bash
   git clone https://github.com/sheima-ayadi/web_vitrine_front.git
   ```
2. Ouvrez le fichier `index.html` dans votre navigateur préféré.

## ✨ Fonctionnalités

- Navigation fluide entre les différentes sections.
- Support du **Mode Sombre / Mode Clair** (automatique selon le système ou manuel).
- Design moderne et responsive.
- Sections dédiées pour chaque service (Cybersécurité, Dev Web, ERP, Microsoft 365, etc.).
